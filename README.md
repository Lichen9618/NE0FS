# NE0FS

**free** storage service on neo network inspired by the flood attack.

# Usage

## Upload

```sh
./NE0FS -c upload -f data.jpg
```

get the printed `<headerhash>` which can be used to download

## Download

```sh
./NE0FS -c download -s <headerhash> -f dst.jpg
```
